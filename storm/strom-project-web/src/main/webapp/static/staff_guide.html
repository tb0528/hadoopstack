2014年4月4日更新 以后看到更新日期变了，就往下看看内容

1.对于填写过报名表的学员，课程咨询第一时间录入到系统中。如果出现学员告诉多个课程咨询已经缴费时，谁录入的算谁的；传智、黑马等培训机构都是这个规则；

2.对于登录过但是没交费的学员，要经常联系，把交流的内容记录到”访谈记录“中。经常联系，就会熟悉，之后自然缴费的；

3.新的课程咨询，要尽可能快的多增加群里的人数。只有人数多了，缴费的概率才大；

4.对于新入群的会员，除了在群里欢迎之外，还要私聊，了解他的需求，如果有报名意向，就很容易成交。不要等着学员主动找你。用户的需求是要挖掘的；

5.我们是一个团队，可以相互竞争，绝不能相互攻击。有问题，双方协商解决。如果让我做裁判的话，先各打五十大板再说。

6.课程咨询在自己负责的群里招生，尽量避免在其他课程咨询的群里做宣传。可以相互加群，毕竟大家兼职，必要的时候可以相互帮上忙，所以不要在自己群里踢其他课程咨询。

7.在我们发展过程中，会有很多的事情出现，需要有专人去做。希望大家踊跃发现问题，解决问题。牵扯很多精力的，每月都可能做的，那就跟我要补助，我很愿意给。这方面，小兵做的很好，见解很多。

我把自己定位于教学管理，喜欢研究新课程，喜欢把这些东西讲给学生听，喜欢把课程尽可能讲的浅显易懂、深入浅出。我不想做管理、不想做宣传，但是公司发展有这方面需要，希望大家多多承担责任。

咱们年底会有年终奖，人人有份。在工作中，会有很多事情，有的事情做了但是不容易计算薪酬。那就在年终奖里体现吧！

8.我们诚实守信，不要夸大宣传，我们依靠质量取胜，疯狂赚钱，心里踏实，睡觉安稳！

祝我们小小默每位同事心情愉悦、天天开心、多多赚钱！！
